# arte-em-jogos

O que aprendi / Técnicas Utilizadas

Este projeto foi construído sem o uso de frameworks, focando nos fundamentos modernos de HTML, CSS e JavaScript:

- Bento Grid Layout: Construção de uma interface complexa e responsiva utilizando CSS Grid nativo (`grid-template-columns`, `span`), adaptando a experiência visual perfeitamente para dispositivos móveis.
- Intersection Observer: Uso da API nativa do navegador para engatilhar animações de *reveal* nos cards de conteúdo apenas quando entram na tela, otimizando o carregamento e a performance.
- Design Tokens & Dark Mode: Estruturação da paleta de cores e tipografia via Variáveis CSS (`:root`), garantindo consistência visual e um contraste atmosférico ideal para leitura.
- CSS Art e Elementos Visuais: Substituição de assets de imagem por formas criadas via CSS (gradientes radiais/cônicos, `clip-path` e manipulação de bordas), reduzindo o peso da página e demonstrando controle de renderização nativa.
- Acessibilidade e Semântica: Implementação de atributos `aria-*` para o menu mobile e tags semânticas do HTML5 para estruturar as análises de arte.
